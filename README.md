# ChatGPT-Powered Personal Finance Mentor

## Project Description
This project aims to provide users with a personal finance mentorship experience powered by ChatGPT. The mentor will provide users with guidance on budgeting, saving, investing, and other financial topics, helping them to make more informed decisions about their finances.

## Setup Instructions
1. **Clone the Repository**  
   Open your terminal and run the following command to clone the repository:
   ```bash
   git clone https://github.com/ShubhiGoel5/personal-coding-mentor.git
   ```

2. **Install Dependencies**  
   Navigate into the project directory:
   ```bash
   cd personal-coding-mentor
   ```  
   Then, install the necessary dependencies:
   ```bash
   npm install
   ```

3. **Environment Variables**  
   Create a `.env` file in the root directory of the project and add your API keys and any other required configurations.
   Example:
   ```bash
   CHATGPT_API_KEY=your_api_key_here
   ```

4. **Run the Application**  
   You can start the application by running:
   ```bash
   npm start
   ```
   Access the application in your web browser at `http://localhost:3000`
